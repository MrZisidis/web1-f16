TextMate

textmate is a popular text editor for web development. It is non-free but is installed on the MassArt computers.

ATOM

Atom another great text-editor to create and edit all of our HTML and CSS files during web development. First download and install the program. Atom is customizable with *packages*, there are a few you should install to make web development easier (namely auto adding of ending tags and automatic checking of code), see below.

Packages

  • open Atom
  • go to menu Atom - Preferences
  • then click + Install
  • under section Install Packages where it says Search Packages find and install the following packages by copy-pasting the name and hitting enter:
    • atom-beautify
    • autoclose-html
    • autocomplete-plus
    • pretty-json
    • HTMLHint
    • color-picker (press command shift c) to bring up color picker

Sublime Text

Sublime Text is a popular text editor for web development. It is non-free but has a gracious trial period.

Other Text Editors

  • PhpStorm —designed for PHP but quite useful for web development, built-in code validation
  • Brackets —free from Adobe, recently Open-Sourced
  • TextWrangler —non-free, installed on MassArt computers
  • Xcode —the only web development editor for the Mac that is reasonably usable by blind programmers
  • Visual Studio —the only web development editor for Windows that is reasonably usable by blind programmers

Settings

There are few settings worth changing in Atom. From the menu bar, Atom - Preferences…
  • Under Packages search for Whitespace, uncheck Remove Trailing Whitespace

Additional Atom documentation is available on their documentation page.

GitHub Desktop

GitHub Desktop is a graphical interface to github.com. It allows you to easily manage repositories performing commits and pushes to GitHub.

Digital Color Meter

Digital Color Meter is used to sample colors from any on-screen color, from images, websites, icons etc… (Mac Only in Applications/Utilities).

Settings to change:

  1. Change the drop-down to Display in Generic RGB
  2. In the View menu change Display Values to as Hexadecimal

Now to sample a color. Make sure Digital Color Meter is running and is in focus, hover over the color you want to sample and press command shift c, the color value is now in your clipboard, command v to paste into your CSS file.

Note if you click on the window you want to sample then Digital Color Meter will no longer be the focused app and command shift c won't work, always command tab to the Digital Color Meter app before sampling then hit command shift c

WhatFont

WhatFont is a browser extension for Google Chrome that allows you to see which font the web browser is actually using to display a particular text element on the page. This is useful to verify your webfonts are working.

Usage

  1. In Google Chrome visit the webpage for the browser extension and click install
  2. Now visit a webpage and click the WhatFont icon (in the top right-hand corner of the page). Now hovering over text elements will display their font, and clicking them will bring up additional information.

Note that WhatFont only works on http and https pages. It will not work when testing your webpage locally with the file:// protocol.

Adobe Color

Adobe Color (formerly Kuler) is an excellent interactive tool for creating color palettes for your webpages. There is nothing to download, just visit the webpage and start using it. Adobe Color is also available as an iPhone app, with the added feature of generating color palettes from your phone photos.
