Tasks

  1. Create a project description page for your final project.

    • Create a web page describing your final project
    • Link to your final project description page from your student page
    • The project description page should contain the following:
      1. Project description (from assignment #3)
      2. Goal statement (2+ sentences). What do you want to achieve from your website? Why are you building it?
      3. List of proposed content. What will your website contain? An about page? Contact page? Personal information? Personal statement? Images? If images, what kinds of images, what are the images of?
      4. Aesthetic inspiration. Find websites or print media that will help inspire the visual aesthetic of your site. Take photos or screenshots and place (as <img>s) on the project description page. If they are websites be sure to link to them as well.
      5. Style your project description page by re-using your student page CSS file
