Create a student page that will be used to display all of your coursework for the upcoming semester. Due by the start of next class.

Goals

  1. Introduction to HTML tags
  2. Practice using GitHub Desktop
  3. Practice editing code using the Sublime text editor

Tasks

  1. Buy book: HTML and CSS by Jon Duckett. Note that the more expensive hardcover is identical to this 2011 softcover.
  2. Create a free account with Github
  3. Download and install TextMate & Github Desktop, see tools page for links, instructions and setup.
  4. Create your student page:
    1. Fork https://github.com/andrewringler/web1-f16-student
    2. Enable Github Pages in
      Settings | GitHub Pages | Source | Master Branch | Save
    3. After a few minutes your website should be live at https://github.com/your-github-username/web1-f16-student mine is live at https://andrewringler.github.io/web1-f16-student/
  5. Customize your own student page to contain:
    1. Your Name <h1>, <title>
    2. Your email address as a link <a> with mailto:, or a fictional one if you are worried about SPAM.
    3. A list of at least five links to website designs that you like <ul>, <li>, <a>, and a short description explaining why.
    4. A paragraph or two describing why you are taking the class, what you hope to achieve and technologies you are interested in. <p>
    5. One or more screenshots of web pages or sections of web pages that you enjoy <img>
    6. Separate sections with <h2> or <h3> tags
  6. Validate your page using https://validator.w3.org/ and fix any issues.
  7. Sync your changes with GitHub. You may sync as often as you desire.

Notes on index.html

Remember default paths? When you visit https://andrewringler.github.io/web1-f16-student/ your web-browser automatically looks for index.html pages. So https://andrewringler.github.io/web1-f16-student/ and https://andrewringler.github.io/web1-f16-student/index.html are the same page, just with different looking URLs.
