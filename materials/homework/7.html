Homework #7 —CSS Zen Garden

  • Course Home

According to its website, the CSS Zen Garden is “a demonstration of what can be accomplished through CSS-based design.” The website is both a gallery of inspiring examples of web design and a challenge to would be CSS masters. The rules of the challenge are as follows:

  1. Download one HTML file and one CSS file (provided by the site)
  2. Modify the stylesheet in any way you wish, but not the HTML
Due by the start of next class.

Adapted from Saul Baizman's final client project assignment.

Goals

  1. Gain an understanding of the power and vast limitations of influencing visual design by only changing CSS.
  2. Step into a not yet forgotten piece of web design history.

Deliverables

  1. Style your own version of CSS Zen Garden modifying only the CSS
  2. Link to your Zen Garden from your Student Page
