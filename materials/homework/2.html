Homework #2

  • Course Home

Tasks

  1. Continue working on your student page index.html. Make sure it contains the following content. Some content you should already have completed last week
    1. Your Name <h1>
    2. Your preferred email address as a link <a>
    3. A paragraph describing why you are taking the class. 3 or more sentences please. <p>
    4. 3 or more links to website designs you like formatted as a list <ul>,<li>,<a>
    5. 1 or more photos (of you or something else) <img>
    6. A list of class goals. What do you want to get out of the class? 2 or more class goals please. Formatted as as an ordered list <ol>,<li>
    7. A page title <title>

    Separate different sections of your student page with <h2> tags.

  2. CSS Type Styling:
    1. Create a new style sheet for your student page. Name it yourname.css
    2. Make sure you are loading your stylesheet from your index.html page
    3. Style your page! Have some fun. Include any type properties you want, but make sure to include (at a minimum) the following CSS properties:
      • A font choice font-family
      • Type sizes for your h1, h2, p, li tags
      • Margins for 1 or more of your tags margin-top, margin-bottom
      • Margins for your page body margin-top, margin-right, margin-bottom, margin-left
  3. Bonus (optional)
    1. Create an alternate style sheet for your student page. Name it something like yourname-alt.css
    2. Duplicate your index.html page, name it something like index-alt.html and link to your new stylesheet
    3. Make your page look completely different. Play around with styles.
    4. From your original index.html page create a link to your new page so that we can see it.
