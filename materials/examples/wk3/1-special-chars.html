Special Characters

UTF-8

The easiest way to make sure most characters will just work, is to add <meta charset="utf-8"> in the <head> section of our page. 85% of all web pages do this. This will allow us to use UNICODE characters in our page which is a set of over 100,000 characters from hundreds of languages so one may type in any language they desire. Like հայերեն (Armenian) or 中国 (Chinese). As well this is even required to type special characters in english like an — (em dash) or ellipses … or even “proper double quotes.”

Typing special characters

On the Mac there are several ways to insert specials characters, the most common being
  • the `Character Viewer` available in the menu as `Edit - Special Chracters` for example I can type Nike®
  • Hold down the key you are typing for accents for example by holding down `e` I can type André
  • Using keyoard shortcuts like `option ;` for an ellipses … You can see a list of keyboard shortcuts using the keyboard viewer.
  • Copy and paste a special character from any web page into your text editor

HTML Entities With Special Meaning

There are only 4 special characters that are used in HTML itself so if want to display them on a page (for example when showing code samples) we need to use a special form to show them.

  • &lt; represents the < sign.
  • &gt; represents the > sign.
  • &amp; represents the & sign.
  • &quot; represents the " mark. Note that is the upright quote used for coding. It is not the double quotes we use when we are quoting someone. For that we don't need a special tag “no special tags for these quotes.”
