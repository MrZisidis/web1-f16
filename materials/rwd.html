Tips

max-width: 100%;

Make sure to always have 

img {
  max-width: 100%; 
}
in your css file so that images won't escape out of their containers as you resize them.

Ethan's target รท context = result was developed in a time when many web design teams had very separate roles for designers and front-end coders. Designers would often create mockups in Photoshop then hand them over to developers. Now (and especially when working alone) it makes perfect sense to just work with %s directly in your css; forget trying to copy your PSD exactly.

scale & device-width

Almost always you should have the following line in the <head> section of your website. 

<meta name="viewport" content="initial-scale=1.0, width=device-width" />

You should pretty much never add maximum-scale=1 to the above line because that will prevent users from using pinch-to-zoom which many, many, many, many people use (and need) to actually see your website.
